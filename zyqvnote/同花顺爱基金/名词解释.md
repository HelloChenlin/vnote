#### 上行与下行
```
银行卡钱通过支付机构到基金公司
基金公司通过监管行把钱打到银行卡
```
#### 未付收益
```
普通赎回：
    全部赎回会将未付收益全部带出，包括未付收益打到银行卡，当天收益作为份额打到基金账户
    部分赎回不会带出未付收益 ,赎回部分当天仍然收益。
快速赎回：
    全部赎回会存在0.01元，用于发放未付收益，当天无收益
    部分赎回不会带出未付收益，当天无收益
结转：
    计算出当天未付收益后，结转前一天的未付收益，即分红。
计算：
    未付收益是由当天的份额*当天净值*万份收益
```

#### 卡bin
```
BIN号即银行标识代码的英文缩写。
BIN由6位数字表示，出现在卡号的前6位，由国际标准化组织(ISO)分配给各从事跨行转接交换的银行卡组织。
```
