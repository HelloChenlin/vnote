# 排序
```
Idea全称intellij Idea,是集各种语言和插件于一体的开发环境。依靠其中的插件，可以进行包括java,C++,python等语言的开发，被认为是超越了Eclipse的ide。其中的智能，代码自动提示，代码的重构，多环境开发，卓越的ui设计极其优秀。同时支持svn、git、cvs等版本开发工具。IDEA作为东欧jetbrains公司的主要产品，受到广大程序员的热烈追捧，包括阿里，腾讯等公司也在使用。idea的开发者是以严格律己的欧洲程序员为主。当然，像IDEA旗舰版本目前还是收费的，免费版只支持java。
IDEA的特色功能有以下几点：
3.1.1智能工具
当我们想要选取一个方法时，或者一个循环，甚至想要有小到大的从变量开始，最后选取类。IDEA便提供了这种功能，你只需要在默认的设置中选择Ctrl+W,然后确认。就可以实现功能，在系统项目重构时，有着不可忽略的作用。
当我们的代码出现错误时，IDEA能分析代码错误的地方，并以快捷键的方式解决。例如MAVEN获取jar包时输错了版本号，IDEA能通过索引自动修复。
3.1.2丰富的导航模式
在Eclipse中查看过去的记录比较麻烦，需要按三个键，而在IDEA中只需要Ctrl+E就可以做到显示最近使用过的文件，双击shift出现你需要搜索的类名，同时还具有自动补充功能，从此不必担心找不到类。同时，IDEA还支持自动生成类图并且下载到本地，对于了解类的结构有很大帮助。
3.1.3 对版本工具的支持
    IDEA默认支持分布式版本工具Git，集中式版本工具SVN以及Github的连接，极大的方便了安装插件时的烦恼，提高了与github社区的交流。
3.2 MySql
Mysql目前已经渐渐取代oracle，作为数据库的首选。与redis，mogodb等非关系型数据库不同，Mysql是一个极其流行的关系型数据库，由瑞典的AB公司所开发，目前已经属于Oracle公司旗下。Mysql目前主要应用在web方面，在这方面有着极高的性能，被人们誉为web上最好的关系型数据库，与Apache,linux，php组成了经典的LAMP（一个优秀的开发环境）。
关系型数据库即将数据通过不同的表分类，每张表存放不同的数据，而不是将所有数据存放到同一个地方，减少了冗余。通过这种方法能大大提高查询的速度和查询的灵活性。
Mysql具有关系型数据库都有的功能：视图，存储过程,触发器等等。同时，Mysql还拥有Myisam,innodb等存储引擎，用户可根据系统的不同，使用不同的存储引擎提高效率
3.3 Spring
作为java开发中的轻量级开源万能框架，由johnson所开发。Spring几乎占据着java这个生态的半壁江上，各种框架层出不穷：SpringCloud,SpringMVC,SpringData,SpringBoot。它主要解决了Service层和其他层之前的耦合问题。通过面向接口的方式，实现控制反转和依赖注入，由容器通过配置自动创建对象并且注入到某个接口或者抽象类。目前已开源。
3.4 SpringMVC
SpringMVC是在Spring的基础上进行开发的框架，是目前流行的web开发框架组合SSM中中间的一环，其中主要作为控制层进行开发，获取请求，与数据库dao层进行交互，获取数据，最后进行转发。相比于Strus1,Struts2等重量框架，SpringMVC更加轻，同时配置简单易懂。
3.5 Mybatis
MyBatis与hiberate一样，是一种优秀的ORM持久化框架，它支持动态定制sql,在sql中编写存储过程，通过resultmap，与数据库进行高级映射。同时通过注解或者xml配置，Mybatis能极大的避免手动设置参数和结果集的遍历，将结果集合转化成用户自定义的pojo(java中的一种对象),是目前流行的web开发框架组合SSM中重要的一部分。
3.6 Maven
在尚未出现管理工具之前，找寻需要的jar包只能通过互联网上搜寻，而且还可能找到版本不一致的jar包导致整个系统出错。我们希望能够有一种快捷标准化的方式来对项目进行管理，而MAVEN就是这样一个能帮你自动管理项目的一种工具。
MAVEN可以使用Group Id,Artifact Id以及版本号形成一个坐标，从maven中央仓库下载自己所需要的jar包。有时候为了方便，也可以自己创建一个Maven私服用于保存jar包，之后便不需要再从中央仓库进行获取了。
当然，除了jar包的管理外，Maven还可以通过命令的方式进行项目的管理。只需Clean，package,install等命令即可。
Maven同样支持脚本的构建，可重用型较大，所以我们可以看到依赖脚本，maven便能够创建出一个简单的项目。
由于Maven的功能强大，许多公司早已将maven的学习列为必修课。

3.7 Druid
在对数据库进行操作时，必须先创建与数据库的连接，在操作结束后关闭连接，但创建连接是一个比较耗时的操作。当出现大量的并发时，可能会由于连接数过多导致数据库宕机。数据库连接池就是在这种情况下诞生的，它能创建指定的连接数，在系统与数据库交互完毕后不是销毁连接，而是将连接继续放到数据库连接池中，能极大的提高效率。
Druid就是一个数据库连接池，由阿里巴巴开发。作为目前最好的连接池之一，无论是功能还是性能亦或是扩展性方面，都远远超过DBCP,C3P0。Druid已经完全应用在阿里巴巴的应用上，而且经过了像双十一这样流量极其大的活动的考验，值得信赖。
3.8 BootStrap
BootStrap，来源于Twitter,基于CSS,HTML,JavaSrcipt进行开发，是一款极其简单的web前端框架。BootStrap具有简洁、直观、强悍的特点，具有独特的栅格系统，支持响应式，移动设备优先等功能。同时BootStrap还能提供表格，表单，按钮，图片格式切换等风格多变的前端模块。使用者只需要引入BootStrap的开发包，便能够通过短短的几行代码实现这些功能。
作为极受欢迎的前端框架，BootStrap已经在jekyll中文网，Chost中文网，同花顺校招等网站投入使用。
```
